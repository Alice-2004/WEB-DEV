Web development

HTML
-HyperText Markup Language, is the standard language for creating web pages. It provides the structure and content for websites, using tags to define elements like headings, paragraphs, and links. HTML works with CSS for styling and JavaScript for interactivity, forming the foundation of the web.

CSS
-CSS (Cascading Style Sheets) is a language that describes the presentation of a document written in HTML or XML. It controls how web pages look, handling elements like colors, fonts, layout, and more, allowing for a separation of content and presentation.

JavaScript
-JavaScript (JS) is a lightweight, versatile programming language primarily used for web development, adding dynamic behavior to websites. It's a core technology of the web, alongside HTML and CSS. Beyond web pages, JS is also used in environments like Node.js.
